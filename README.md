## ✏️ Como atualizar o conteúdo

| O que editar | Onde encontrar |
|---|---|
| Eventos e palestras | `index.html` → buscar `tab-2025`, `tab-2024`... |
| Projetos | `index.html` → buscar `projeto-card` |
| Formação acadêmica | `index.html` → buscar `formacao-card` |
| Redes sociais | `index.html` → buscar `rede-item` |
| Stack técnica | `index.html` → buscar `tech-tag` |
| Cores e tipografia | `styles.css` → seção `TOKENS` (variáveis `:root`) |
| Animações e interações | `script.js` |
